# Практическая работа №2: Параллельная сортировка

## Описание
Программа реализует и сравнивает последовательные и параллельные версии алгоритмов сортировки:
- Пузырьковая (Bubble Sort) -> Параллельная четно-нечетная (Odd-Even Sort)
- Выбором (Selection Sort) -> Параллельный поиск минимума
- Вставкой (Insertion Sort) -> Оставлена последовательной (из-за сложности параллелизации)

## Как запустить

### 1. Компиляция
В папке `practice2` выполните:

```bash
g++ -fopenmp sorting.cpp -o sorting
```

### 2. Запуск
```powershell
.\sorting.exe
```
(или `./sorting` на Linux/macOS)

Программа автоматически протестирует сортировки на массивах размером 1000, 10000 и 30000 элементов и выведет время выполнения.
